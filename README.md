# Neon District (sandbox 3D original)

Prototipo de juego 3D estilo sandbox urbano hecho desde cero, con contenido **original** para evitar uso de marcas o material con copyright de franquicias existentes.

## Incluye

- Mapa urbano 3D con iluminación, niebla y edificios procedurales.
- Movimiento en primera persona (WASD + Shift).
- Arma de pulso con sistema de impacto por raycast.
- Enemigos hostiles con IA básica de persecución.
- Sistema de misiones encadenadas.

## Ejecutar

Abre `index.html` en un navegador moderno o sirve la carpeta con un servidor local:

```bash
python3 -m http.server 4173
```

Luego abre `http://localhost:4173`.
